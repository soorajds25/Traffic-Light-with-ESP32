# Traffic Light Control System Using Two ESP32

Imagine a situation where you want to cross a busy highway with no intersections, just a pedestrian cross with no vehicles slowing down for you. That is the exact problem we are solving here. In this system we have **_two traffic lights_**, one for the pedestrian and one for the vehicles. The traffic signal of the **vehicles** is controlled by the **master ESP32** whereas the traffic signal of the **pedestrians** is controlled by the **slave ESP32**.
